# <img src="https://github.com/user-attachments/assets/e13339da-a0ca-4d21-b53f-c693937ba272" height="19" alt="trading icon"> Testing of the Capital.com web application

Во время интенсива SandBox я погрузился в реальный процесс тестирования сайта Capital.com!
Ежедневно проводил ad‑hoc‑тестирование отдельных разделов сайта - с фокусом на конкретный язык интерфейса и лицензию. В обязанности входило:
<ul>
  <li> подготовка детализированных <a href="https://docs.google.com/spreadsheets/d/1FFMJEg57SFY2b8eqZbzXaq0O3lre_H4SToXu2zd8ug8/edit?usp=sharing"> баг‑репортов </a> ; </li>
  <li> составление <a href="https://drive.google.com/drive/folders/17HQOev6OTY9mNe81xdCKG0g0gL6nYbt6?usp=sharing"> еженедельных отчётов </a> по результатам тестирования; </li>
  <li> совместная с командой работа над <a href="https://docs.google.com/spreadsheets/d/1U69dhpNgv3_hUSugEVqv8b-Ylkleoz57u86MTbIKdVc/edit?usp=sharing"> smoke‑checklist </a> для ключевых разделов сайта. </li>  
  <li> выполнение практического <a href="https://docs.google.com/document/d/1lUMp-bjflW7DL55RtuW7UnnH7Oolpu6skk-MefNYjOA/edit?usp=sharing"> задания по SQL </a> ; </li>
  <li> разработка <a href="https://drive.google.com/drive/folders/1WTK4RwhPP_ZpeRkVCUw9VhlKSR0dQ2Dg?usp=sharing"> мини‑проекта </a> — HTML‑страницы, посвящённой личному хобби; </li>
</ul>
Опыт дал мне отличную практику в QA и укрепил навыки командной работы.
